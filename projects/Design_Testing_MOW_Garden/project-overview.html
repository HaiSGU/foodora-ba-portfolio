Design & Testing — MOW Garden (Course Project)

System design + test planning + test design + JMeter performance testing for a garden e-commerce web app

Note: This is a university/course project deliverables pack. The focus is on design and testing artifacts (not a production deployment).

Key Artifacts

Decomposition (PNG) Shopping Cart (PNG) Test Plan (PDF) JMeter PERF-01 Full Deliverables

Objective

Build an end-to-end design & testing pack for a small e-commerce product scope (MOW Garden), including core module design views (decomposition + sequence diagrams) and a structured testing approach (test plan, test design, and JMeter performance test plans).

The Problem Framing

Quality Risk

E-commerce apps often fail not only on missing features, but on quality gaps: inconsistent flows, weak access control, fragile cart/checkout logic, and performance issues under load. This project focuses on making core flows clear, testable, and measurable.

OUT OF SCOPE

  • Real payment gateway integration
