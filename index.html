Overview

Foodora is a portfolio case study for a pickup-only ordering MVP that helps local restaurants reduce third-party delivery fees by enabling direct ordering and pickup scheduling.

This repository contains BA deliverables (PRD, use cases, BPMN, ERD, UAT package, traceability) presented in a clean, static website format.
